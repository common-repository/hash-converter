== Description ==

'Hadsh Converter' is a Hashing Power Calculator.

Just type a digit and select hash unit to calculate and click on calculate button.

== Installation ==

Just download and activate 'Hash Converter'.

Use this shortcode to show Hash Converter On your page - '[AWL-HC]'
